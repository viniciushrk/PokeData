# PokeData

## Roteiro CRUD
- Pokemon (cadastrar, listar, atualizar, deletar) 
	
* Cadastrar Pokemon
	* Nome
	* Tipo
	* Imagem
	* Descrição
	* Geração
	* Stars
* Listar Pokemon
	* Usar Cache
	* Filtrar por Tipo
	* Filtrar por geração
* Atualizar Pokemon
	* Stars
	* E seus atributos
* Deletar Pokemon
	* Deletar pokemon 
	* Remover stars
